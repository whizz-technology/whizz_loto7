# Whizz loto7について
  
***
　Whizz loto7は、PHP初心者向け学習教材です  
　宝くじのロト7の数字を予想するサイトを作成します
***  
  
### ロト7とは
> 1～37の数字の中から異なる7個の数字を選ぶ、数字選択式宝くじです。    
  
### 学べること
* 言語
  * PHP
  * HTML・CSS
* 処理
  * 条件分岐（if）  
  * 反復処理（for,foreach)  
  * 配列
  
### 入っているもの
* img　…使用する画像3枚、完成イメージ1枚   
* common.css  
* loto7.php　…学習ソース  
* loto7_complete.php　…完成ソース  
* README  
* LICENSE
  
### 必要なもの
* PHP実行環境

### 実装内容
更新する度に、下記3点がランダムで表示される処理を実装します  
* あなたの金運  
  →  1～5つの★をランダム表示する
* あなたのラッキー方位  
  →東西南北をランダムで表示する
* 7つの抽選番号  
  →1～37の数字を重複なく、ランダムで表示する
